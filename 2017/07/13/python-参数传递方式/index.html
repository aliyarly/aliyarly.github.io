python_参数传递方式

python中一切皆为对象

对象

数字是对象，列表是对象，函数也是对象，任何东西都是对象

变量

变量是对象的一个引用（又称为名字或者标签），对象的操作都是通过引用来完成的

变量本身没有类型信息，类型信息存储在对象中

赋值操作 = 就是把一个名字绑定到一个对象上。就像给对象添加一个标签

函数参数

本质上应该是传递的对象，根据对象的类型来确定传递的方式：

  • 如果是不可变的对象则可以理解为按值传递
  • 是可变对象则可以理解为按引用传递
