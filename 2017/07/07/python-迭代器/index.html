  1. 可迭代的对象

能够循环遍历访问对象的各个元素，能够使用for ... in ...,所有的数据依次加载进内存中，且对象是可重复使用
例如：list, turple,dict,set等

对象的必须实现__iter__方法，将相应的对象转换为可迭代对象，内置数据结构是包含的．

  2. 迭代器优点

不必事先准备好整个迭代过程中所有的元素。迭代器仅仅在迭代到某个元素时才计算该元素，而在这之前或之后，元素可以不存在或者被销毁。适用于大量数据．

提供了一个统一的访问集合的接口，只要定义了__iter__()方法对象，就可以使用迭代器访问
